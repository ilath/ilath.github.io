候选样式表

将rel属性的值置为alternate stylesheet，就可以定义候选样式表，只有在用户选择这个样式表时才会用于文档表现。如果为一个rel为styleesheet的link指定了title，也就指定了该样式表要作为首选样式表。如果没有为样式表指定title，那么它将作为一个永久样式表，始终作用于文档的显示。

<link rel="stylesheet" type="text/css" href="sheet.css" title="Default"/>

<link rel="alternate stylesheet" type="text/css" href="sheet1.css"/>

属性选择器

简单属性选择

  • 简单属性选择器

    h1[class] { color: sliver; }

  • 元素属性选择器

    img[alt] { border: 3px solid red; }

    *[title] { font-weight: bold; }

    a[href][title] { font-weight: bold; }

根据具体属性值选择

  • 根据属性及其值选择元素

    这种格式要求必须与属性值完全匹配

    a[href="http://www.example.org"] { font-weight: bold; }

    a[href="http://www.example.org"][title="example"] { font-weight: bold; }

根据部分属性值选择

  • 根据部分属性值选择元素

    ~ 根据属性值中出现的一个用空格分隔的词来完成选择

    p[class~="warning"] { font-weight: bold; }

    这个选择器构造等价于点号类名记法。 p.warning 和 p[class~=”waring”] 是等价的。

  • 子串匹配属性选择器

    1. ^ [foo^="bar"]

      选择foo属性值以“bar”开头的所有元素

    2. $ [foo$="bar"]

      选择foo属性值以“bar”结尾的所有元素

    3. * [foo*="bar"]

      选择foo属性值中包含子串“bar”的所有元素

    其他用法示例：

    根据属性的值悬着元素

    a[href*="oreilly.com"] { font-weight: bold; }

特定属性选择类型

  • [att|="val"]

    这个规则会选择 att 属性等于val或者以 val- 开头的所有元素。

后代选择器

  • 选择子元素

    > 子结合符

    h1 > strong { color: red; }

    这个规则会把第一个h1下面出现的strong元素变成红色，但是第二个h1下出现的strong元素不会受影响。

  • 选择相邻兄弟元素

    + 相邻兄弟结合符

    h1 + p { margin-top: 0; }

    用一个结合符只能选择两个相邻兄弟中的第二个元素。另外，两个元素之间的文本内容不会影响相邻兄弟结合符起作用。
